# Show Stats Skill

Display task statistics and the full task list.

## Arguments

None required. `$ARGUMENTS` is ignored.

## Steps

1. Run: `python -m task_manager stats`
2. Run: `python -m task_manager list`
3. Summarize the output in one sentence (e.g., "You have 3 pending tasks: 1 high, 1 medium, 1 low priority.")

## Example Usage

```
/show-stats
```

## What It Shows

- Total tasks created
- Pending vs completed count
- Breakdown by priority
- Full task list with IDs, titles, priorities, and status
